## Gemeinsame Verbesserungsdimensionen

Jeder Prompt wird auf folgende Dimensionen geprüft:

1. **Clarity** - Ist die Struktur klarer geworden?
2. **Specificity** - Sind Anweisungen präziser?
3. **Anti-Hallucination** - Sind Sicherheitsmaßnahmen stärker?
4. **Token Efficiency** - Ist der Prompt kürzer/prägnanter?
5. **Edge Cases** - Werden mehr Randfälle abgedeckt?
6. **User Experience** - Ist die Interaktion verbessert?
